# uTools 指令

> 指令格式：前缀(1位) + 命令 + . + 后缀
> 例：`$mail.163` — `$` 前缀，`mail` 命令，`163` 后缀
> `@` = 工作信息，`$` = 个人信息

## 个人信息

| 关键词 | 指令 | 参数 | 说明 |
|---|---|---|---|
| 姓名/名字 | `$name` | | 获取姓名 |
| 手机/电话/手机号 | `$phone` | | 获取个人手机号 |
| 邮箱/个人邮箱 | `$mail` | `.163` / `.gmail` | 获取个人邮箱，默认 163 |
| 工号/员工编号 | `$id` | | 获取工号 |
| 日期/今天/当前日期 | `$date` | | 获取当前日期（YYYY-MM-DD） |

## 工作信息

| 关键词 | 指令 | 参数 | 说明 |
|---|---|---|---|
| 工作邮箱/公司邮箱 | `@mail` | `.cyg` / `.gmail` | 获取工作邮箱，默认 cyg |
| 员工编号 | `@id` | | 获取员工编号 |
| 日期 | `@date` | | 获取当前日期 |

## CRM 自动化

| 关键词 | 指令 | 说明 |
|---|---|---|
| 更新数据/触发工作流 | `utf` | 更新数据并触发工作流 |
| ID查询/按ID查 | `fbi` | 根据数据 ID 查询 |
| 条件查询/查一条 | `fo` | 根据条件查询一条数据 |
| 查询全部/列表查询 | `fnd` | 查询满足条件的所有数据 |
| 聚合/统计 | `agg` | 聚合指定字段 |
| 分组聚合/分组统计 | `abg` | 按分组字段聚合（分组 + 聚合） |
