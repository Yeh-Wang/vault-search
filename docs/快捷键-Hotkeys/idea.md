# IDEA

## 编辑

| 关键词 | 快捷键 | 说明 |
|---|---|---|
| 补全/代码补全 | `Ctrl + Space` | 基本代码补全 |
| 智能补全 | `Ctrl + Shift + Space` | 根据预期类型过滤 |
| 补全语句 | `Ctrl + Shift + Enter` | 自动补全语句 |
| 参数提示 | `Ctrl + P` | 参数信息提示 |
| 文档/快速文档 | `Ctrl + Q` | 快速文档查看 |
| 外部文档 | `Shift + F1` | 外部文档 |
| 悬停/简要信息 | `Ctrl + 鼠标悬停` | 简要信息 |
| 错误描述 | `Ctrl + F1` | 显示光标处错误或警告描述 |
| 生成代码/getter/setter | `Alt + Insert` | 生成代码（Getter/Setter/构造器等） |
| 重写方法/override | `Ctrl + O` | 重写方法 |
| 实现方法/implement | `Ctrl + I` | 实现方法 |
| 包围/try/catch/if | `Ctrl + Alt + T` | 包围代码块（if..else/try..catch 等） |
| 注释/行注释 | `Ctrl + /` | 行注释 |
| 块注释 | `Ctrl + Shift + /` | 块注释 |
| 扩大选择 | `Ctrl + W` | 逐步扩大选择 |
| 缩小选择 | `Ctrl + Shift + W` | 逐步缩小选择 |
| 上下文信息 | `Alt + Q` | 上下文信息 |
| 快速修复/意图 | `Alt + Enter` | 显示意图操作和快速修复 |
| 格式化 | `Ctrl + Alt + L` | 格式化代码 |
| 优化import/导入 | `Ctrl + Alt + O` | 优化 import |
| 自动缩进 | `Ctrl + Alt + I` | 自动缩进 |
| 缩进 | `Tab` / `Shift + Tab` | 缩进/取消缩进 |
| 剪切 | `Ctrl + X` | 剪切当前行 |
| 复制 | `Ctrl + C` | 复制当前行 |
| 粘贴 | `Ctrl + V` | 粘贴 |
| 历史粘贴 | `Ctrl + Shift + V` | 从历史剪贴板选择粘贴 |
| 复制行 | `Ctrl + D` | 复制当前行 |
| 删除行 | `Ctrl + Y` | 删除光标所在行 |
| 合并行 | `Ctrl + Shift + J` | 智能合并行 |
| 拆分行 | `Ctrl + Enter` | 智能拆分行 |
| 新行 | `Shift + Enter` | 开始新行 |
| 大小写/转大小写 | `Ctrl + Shift + U` | 切换大小写 |
| 代码块选择 | `Ctrl + Shift + ]` / `[` | 选择到代码块结束/开始 |
| 删词 | `Ctrl + Delete` / `Backspace` | 删除到单词结尾/开头 |
| 关闭标签页 | `Ctrl + F4` | 关闭当前标签页 |
| 撤销 | `Ctrl + Z` | 撤销 |
| 反撤销/重做 | `Ctrl + Shift + Z` | 反撤销 |
