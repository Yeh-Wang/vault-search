# Obsidian

> [!tip]- 中英字符转化
> 连续输入全角符号自动转化为半角符号
> e.g: 》》 -->  >

## 文件操作

| 关键词 | 快捷键 (Win) | 快捷键 (Mac) | 说明 |
|---|---|---|---|
| 新建/创建笔记 | `Ctrl + N` | `⌃ + N` | 创建新笔记 |
| 时间戳笔记 | `Ctrl + M` | `⌃ + M` | 创建以时间戳为标题的笔记 |
| 新建文件夹 | `Alt + F` | `⌃ + F` | 创建新文件夹 |
| 删除文件 | `Ctrl + D` | `⌘ + D` | 删除当前文件 |
| 搜索/全局搜索 | `Ctrl + Shift + F` | `⇧ + ⌘ + F` | 全局搜索 |
| 切换文件/快速打开 | `Ctrl + K` | | 快速切换文件 |
| 侧边栏 | `Ctrl + R` | `⌘ + R` | 打开/折叠右侧侧边栏 |
