# VS Code

## 通用

| 关键词 | 快捷键 | 说明 |
|---|---|---|
| 命令面板/命令 | `Ctrl + Shift + P` / `F1` | 显示命令面板 |
| 快速打开/跳转文件 | `Ctrl + P` | 快速打开/跳转到文件 |
| 新建窗口 | `Ctrl + Shift + N` | 新建窗口 |
| 关闭窗口 | `Ctrl + Shift + W` | 关闭窗口 |
| 设置 | `Ctrl + ,` | 打开用户设置 |
| 快捷键设置 | `Ctrl + K` `Ctrl + S` | 查看或修改键盘快捷键 |

## 基础编辑

| 关键词 | 快捷键 | 说明 |
|---|---|---|
| 剪切行 | `Ctrl + X` | 剪切整行（无选中内容） |
| 复制行 | `Ctrl + C` | 复制整行（无选中内容） |
| 上移行/下移行 | `Alt + ↑` / `↓` | 向上/向下移动整行 |
| 复制行上/复制行下 | `Shift + Alt + ↓` / `↑` | 向上/向下复制整行 |
| 删除行 | `Ctrl + Shift + K` | 删除整行 |
| 插入行下 | `Ctrl + Enter` | 在下方插入新行 |
| 插入行上 | `Ctrl + Shift + Enter` | 在上方插入新行 |
| 匹配括号 | `Ctrl + Shift + \` | 跳转到匹配的括号 |
| 缩进/取消缩进 | `Ctrl + ]` / `[` | 增加/减少缩进 |
| 行首/行尾 | `Home` / `End` | 跳转到行首/行尾 |
| 文件开头/文件末尾 | `Ctrl + Home` / `End` | 跳转到文件开头/末尾 |
| 滚动一行 | `Ctrl + ↑` / `↓` | 向上/向下滚动一行 |
| 滚动一页 | `Alt + PgUp` / `PgDn` | 向上/向下滚动一页 |
| 折叠/展开代码 | `Ctrl + Shift + [` / `]` | 折叠/展开代码块 |
| 折叠子区域 | `Ctrl + K` `Ctrl + [` / `]` | 折叠/展开所有子区域 |
| 全部折叠/全部展开 | `Ctrl + K` `Ctrl + 0` / `J` | 折叠/展开全部区域 |
| 行注释/注释 | `Ctrl + /` | 切换行注释 |
| 块注释 | `Shift + Alt + A` | 切换块注释 |
| 自动换行/换行 | `Alt + Z` | 切换自动换行 |

## 导航

| 关键词 | 快捷键 | 说明 |
|---|---|---|
| 全局符号/符号搜索 | `Ctrl + T` | 显示所有符号（全局） |
| 跳转行/跳行 | `Ctrl + G` | 跳转到指定行 |
| 跳转文件 | `Ctrl + P` | 跳转到文件 |
| 文件符号 | `Ctrl + Shift + O` | 跳转到当前文件符号 |
| 问题面板/错误 | `Ctrl + Shift + M` | 显示"问题"面板 |
| 下一个错误 | `F8` / `Shift + F8` | 下一个/上一个错误或警告 |
| 编辑器历史 | `Ctrl + Shift + Tab` | 在编辑器组历史间切换 |
| 后退/前进 | `Alt + ←` / `→` | 后退/前进 |

## 搜索与替换

| 关键词 | 快捷键 | 说明 |
|---|---|---|
| 查找/搜索 | `Ctrl + F` | 查找 |
| 替换 | `Ctrl + H` | 替换 |
| 下一个/上一个匹配 | `F3` / `Shift + F3` | 查找下一个/上一个 |
| 全选匹配 | `Alt + Enter` | 选中所有查找匹配项 |
| 逐个选中 | `Ctrl + D` | 将下一个匹配项加入选中 |
| 跳过选中 | `Ctrl + K` `Ctrl + D` | 将上一处选择移至下一匹配 |
| 大小写/正则/全词 | `Alt + C` / `R` / `W` | 切换大小写敏感/正则/全词匹配 |

## 多光标与选择

| 关键词 | 快捷键 | 说明 |
|---|---|---|
| 多光标/插入光标 | `Alt + Click` | 在点击位置插入光标 |
| 上方光标/下方光标 | `Ctrl + Alt + ↑` / `↓` | 在上/下方插入光标 |
| 撤销光标 | `Ctrl + U` | 撤销最后一次光标操作 |
| 行尾光标 | `Shift + Alt + I` | 在选中每行末尾插入光标 |
| 选中行 | `Ctrl + L` | 选中当前整行 |
| 选中所有相同 | `Ctrl + Shift + L` | 选中所有当前选择出现处 |
| 选中所有单词 | `Ctrl + F2` | 选中所有当前单词出现处 |
| 扩大选择/缩小选择 | `Shift + Alt + →` / `←` | 扩大/缩小选择范围 |
| 列选择/框选 | `Shift + Alt + Drag` | 列（框）选择 |
| 列选择翻页 | `Ctrl + Shift + Alt + PgUp/PgDn` | 列选择向上/向下翻页 |
